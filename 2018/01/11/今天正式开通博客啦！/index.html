    Welcome to XiaoHHei!

    经过一天的整理终于通过github和Hexo搭好了自己的静态博客，以后闲来没事过来写写东西也是一种消遣，哈哈哈…

    简单介绍一下建博的过程吧

    创建网址
    • 注册个人github账号
    • 在github中创建个人网站的仓库，格式如：yourgithubname.github.io
    环境搭建
    • 下载安装node.js和git
    • 运行git bash输入ssh-keygen -t rsa,回车三下获取密钥文件id_rsa和公钥文件id_rsa.pub
    • 复制公钥文件中所有内容添加到github上进行SSH授权，添加后在git bash中输入ssh -T git@github.com进行测试是否成功
    设置本地博客配置
    • 任意地方创建你的空博客文件夹，右键进入git bash，输入npm install -g hexo安装hexo（回车后等待即可）
    • 输入npm install hexo –save这个时候你会看到命令行出现了一堆白字，紧接着输入hexo v查看是否安装成功
    • 继续输入hexo init初始化hexo
    • 下载好了，再输入hexo s

    这时候我们可以打开浏览器输入 http://localhost:4000/ ,我们的博客就建成功了

    有点糙，详细教程请参考结合hexo在GitHub上搭建个人博客——全过程,嘻嘻…
